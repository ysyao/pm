# 网络设备

信息在网络中的传输主要有**以太网技术**和**网络交换技术**。网络交换是指通过一定的设备，如交换机等，将不同的信号或者信号形式转换为对方可识别的信号类型从而达到通信目的的一种交换形式。常见的有数据交换、报文交换、线路交换和分组交换。按照交换层次的不同，网络交换可以分为物理层交换、链路层交换、网络层交换、传输层交换和应用层交换。

基于无线网络的设备有无线网卡、无线AP、无线网桥和无线路由器等。
